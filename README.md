## 选择哪种缓存方案

在优化系统时，我们一般会考虑到缓存的使用。而现有的缓存方案不外乎就是本地缓存、远程缓存，至于选择哪一种缓存作为方案，标准答案就是看情况。这个就比较尴尬了，这种答案最不利于团队成员的开发。大家都看情况，肯定是各种情况。
